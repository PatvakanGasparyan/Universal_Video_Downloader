# Contributing to Universal Video Downloader

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/universal-video-downloader.git`
3. Create a branch: `git checkout -b feature/my-feature`
4. Set up the development environment (see README.md)

## Development Workflow

1. Make your changes with clear, focused commits
2. Run tests: `pytest --cov=backend`
3. Run linter: `ruff check backend tests`
4. Push and open a Pull Request

## Code Style

- Python 3.13+ with type annotations
- Follow existing project structure and naming conventions
- Keep changes focused and minimal
- Add tests for new functionality

## Pull Request Guidelines

- Fill out the PR template completely
- Ensure CI passes
- Update documentation if needed
- One feature or fix per PR when possible

## Reporting Issues

Use the issue templates for bug reports and feature requests.
