# Browser cookies for yt-dlp (Netscape format)
#
# 1. Install a browser extension such as "Get cookies.txt LOCALLY"
# 2. Log in to YouTube in your browser
# 3. Export cookies for youtube.com as cookies.txt
# 4. Save the file HERE as:  config/cookies.txt
#
# IMPORTANT: cookies.txt contains your login session — never commit it to git.
#
# Alternative path: set COOKIES_FILE in .env or configure via Settings page in the UI.
